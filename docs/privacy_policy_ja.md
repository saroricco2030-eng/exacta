# Exacta プライバシーポリシー

**最終更新日: 2026年4月10日**

Exacta(以下「本アプリ」)はユーザーのプライバシーを尊重します。本ポリシーは、本アプリがユーザー情報をどのように扱うかを明確に説明します。

## 1. 基本原則: 完全ローカル保存

本アプリは**ユーザーデータを1バイトたりとも外部サーバーへ送信しません**。ユーザーが撮影したすべての写真、メモ、GPS座標、プロジェクト情報、ハッシュチェーンは、ユーザー端末の内部ストレージにのみ保存されます。

本アプリは以下を使用しません:
- ユーザーアカウント / 会員登録 / ログイン
- バックエンドサーバー / クラウド同期
- 広告SDK / トラッカー / 分析ツール
- 第三者へのデータ共有・販売

## 2. 収集する情報(端末内部にのみ保存)

ユーザーの同意の下、本アプリは以下の情報を端末内部でのみ処理します:

### 2-1. カメラデータ
- **目的**: 写真・動画撮影
- **保存場所**: 端末内部ストレージ(`/Android/data/com.exacta.app/`)
- **外部送信**: なし

### 2-2. 位置情報(オプション)
- **目的**: 写真スタンプにGPS座標・住所を合成
- **保存場所**: 写真ピクセル内部 + 端末SQLite DB
- **外部送信**: なし
- **備考**: セキュア撮影モードを有効にすると、位置情報を一切収集・保存しません。

### 2-3. 写真ライブラリへのアクセス
- **目的**: スタンプ用ロゴ画像の選択、撮影した写真をシステムギャラリーに追加(任意)
- **外部送信**: なし

### 2-4. マイク(動画撮影時のみ)
- **目的**: 動画の音声録音
- **保存場所**: 端末内部の動画ファイルにのみ含まれる
- **外部送信**: なし

## 3. ユーザーが明示的に共有する場合

以下の場合のみ、ユーザーデータが端末外部に送出されます:
- 「共有」ボタンをタップ → ユーザーが選んだアプリ(LINE、メール、メッセージなど)に送信
- 「エクスポート」メニュー → ZIP/PDFを生成しユーザーが保存先を選択

これらすべての場合、システムの共有シートを経由し、本アプリが任意に送信することはありません。

## 4. データ保持と削除

- すべてのデータは、ユーザーがアプリ内で直接削除するまで端末内部に保持されます。
- アプリをアンインストールすると、すべての写真・メタデータ・ハッシュチェーンが端末から即座に削除されます。
- 本アプリは別途のデータ保持期間を設けず、ユーザーがデータの唯一の所有者です。

## 5. 子供のプライバシー保護

本アプリは13歳未満の子供を対象としておらず、子供から意図的に個人情報を収集しません。

## 6. セキュリティ

本アプリは以下のセキュリティ対策を適用します:
- コード難読化(release build obfuscation)
- ProGuard / R8 コード縮小
- セキュア撮影モード: EXIF位置情報の完全削除 + 別フォルダ分離
- SHA-256ハッシュチェーンによる写真の整合性検証
- NTP時刻同期による端末時計改ざんの防止

## 7. ユーザーの権利

ユーザーはいつでも:
- アプリ内で写真を直接削除する権利
- アプリ自体をアンインストールしてすべてのデータを永久削除する権利
- アプリが要求した権限をシステム設定から取り消す権利

## 8. 本ポリシーの変更

本ポリシーは変更される場合があり、変更時は本ページの「最終更新日」が更新されます。
