# Exacta 개인정보 처리방침

**최종 업데이트: 2026년 4월 10일**

Exacta(이하 "본 앱")는 사용자의 개인정보를 소중히 여기며, 본 처리방침은 본 앱이 사용자 정보를 어떻게 다루는지 명확히 안내합니다.

## 1. 핵심 원칙: 완전 로컬 저장

본 앱은 **단 한 byte의 사용자 데이터도 외부 서버로 전송하지 않습니다**. 사용자가 촬영한 모든 사진, 메모, GPS 좌표, 프로젝트 정보, 해시 체인은 사용자 단말 내부 저장소에만 보관됩니다.

본 앱은 다음을 사용하지 않습니다:
- 사용자 계정 / 회원가입 / 로그인
- 백엔드 서버 / 클라우드 동기화
- 광고 SDK / 추적기 / 분석 도구
- 제3자 데이터 공유 / 판매

## 2. 수집하는 정보 (단말 내부에만 저장)

본 앱은 사용자 동의하에 다음 정보를 단말 내부에서만 처리합니다:

### 2-1. 카메라 데이터
- **목적**: 사진/영상 촬영
- **저장 위치**: 단말 내부 저장소(`/Android/data/com.exacta.app/`)
- **외부 전송**: 없음

### 2-2. 위치 정보 (선택)
- **목적**: 사진 스탬프에 GPS 좌표·주소 합성
- **저장 위치**: 사진 픽셀 내부 + 단말 SQLite DB
- **외부 전송**: 없음
- **참고**: 보안 촬영 모드를 활성화하면 위치 정보를 일절 수집/저장하지 않습니다.

### 2-3. 사진 라이브러리 접근
- **목적**: 스탬프용 로고 이미지 선택, 촬영본을 시스템 갤러리에 추가(선택 가능)
- **외부 전송**: 없음

### 2-4. 마이크 (영상 촬영 시에만)
- **목적**: 영상 음성 녹음
- **저장 위치**: 단말 내부 영상 파일에만 포함
- **외부 전송**: 없음

## 3. 사용자가 명시적으로 공유할 때

다음 경우에만 사용자 데이터가 단말 외부로 나갑니다:
- "공유" 버튼 탭 → 사용자가 직접 선택한 앱(카카오톡, 이메일, 메시지 등)으로 사진/PDF 전송
- "내보내기" 메뉴 → ZIP/PDF 파일 생성 후 사용자가 선택한 경로로 저장

이 모든 경우는 사용자의 명시적 행동 + 시스템 공유 시트를 거치며, 본 앱이 임의로 전송하지 않습니다.

## 4. 데이터 보존 및 삭제

- 모든 데이터는 사용자가 앱 내부에서 직접 삭제할 때까지 단말 내부에 유지됩니다.
- 앱을 제거하면 모든 사진·메타데이터·해시 체인이 단말에서 즉시 삭제됩니다.
- 본 앱은 별도의 데이터 보존 기간을 두지 않으며, 사용자만이 데이터의 유일한 소유자입니다.

## 5. 어린이 개인정보 보호

본 앱은 13세 미만 어린이를 대상으로 하지 않으며, 어린이로부터 의도적으로 개인정보를 수집하지 않습니다.

## 6. 보안

본 앱은 다음 보안 조치를 적용합니다:
- 코드 난독화(release build obfuscation)
- ProGuard / R8 코드 축소
- 보안 촬영 모드: EXIF 위치 정보 완전 제거 + 별도 폴더 격리
- SHA-256 해시 체인을 통한 사진 무결성 검증
- NTP 시각 동기화로 기기 시계 조작 방지

## 7. 사용자 권리

사용자는 언제든지:
- 앱 내부에서 사진을 직접 삭제할 권리
- 앱 자체를 제거하여 모든 데이터를 영구 삭제할 권리
- 앱이 요청한 권한을 시스템 설정에서 철회할 권리

## 8. 본 처리방침 변경

본 처리방침은 변경될 수 있으며, 변경 시 본 페이지의 "최종 업데이트" 날짜가 갱신됩니다.
