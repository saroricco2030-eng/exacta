Exacta는 현장 엔지니어, 시공·점검 기술자, 부동산·보험·물류 실무자를 위한 위변조 불가 사진 기록 도구입니다.

촬영 순간 시간·GPS·주소가 사진 픽셀에 직접 합성되고, SHA-256 해시 체인으로 모든 사진이 수학적으로 연결됩니다. 단 1바이트만 변경되어도 즉시 탐지됩니다.

■ 핵심 기능
━━━━━━━━━━━━━━━━━

▸ 픽셀 번인 스탬프
시간·날짜·주소·GPS 좌표·고도·방위·속도가 사진 내부 픽셀에 합성됨. EXIF 메타데이터 제거로도 절대 지울 수 없음.

▸ NTP 시각 보정
인터넷 표준 시간 서버와 동기화하여 기록. 기기 시계를 임의로 변경해도 실제 촬영 시각이 그대로 남음.

▸ SHA-256 해시 체인 (위변조 탐지)
각 사진은 직전 사진의 해시와 수학적으로 연결됨. 사진 1장만 조작해도 이후 모든 사진의 체인이 깨져서 즉시 발각됨.

▸ 보안 촬영 모드
GPS·주소를 완전 차단하고 EXIF 위치 정보를 스트리핑. 민감 현장(군사·의료·법정 등)에서 안전하게 촬영.

▸ 프로젝트별 자동 분류
프로젝트를 만들고 사진을 자동 분류. 프로젝트별 일괄 공유, 컬러 라벨, 진행/완료 상태 관리.

▸ 법적 증거 팩 PDF 내보내기
사건명·작성자 입력 → 표지·전체 사진·해시·외부 검증 안내가 포함된 PDF 자동 생성. 법정·보험·하자 분쟁 제출용.

▸ 현장 친화 UI
40~50대 기술자를 위한 굵은 글씨, 큰 셔터 버튼(66dp), 장갑 착용 환경 고려한 56dp 터치 타겟. 한 손 조작 가능.

▸ 완전 오프라인
서버 통신 없음. 광고 없음. 계정 없음. 사진은 단말 내부에만 저장됨.

▸ 다국어 지원
한국어 / English / 日本語
