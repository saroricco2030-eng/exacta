## 권장 스크린샷 5장 (스토리텔링 순서)

### 1. 메인 카메라 화면 (Hero Shot)
- **촬영 대상**: 카메라 뷰파인더 + 스탬프 오버레이가 보이는 상태
- **메시지**: "시간·위치를 사진에 새깁니다"
- **팁**: 실제 야외 풍경 위에 스탬프가 깔끔하게 보이도록

### 2. 갤러리 화면 (검증)
- **촬영 대상**: 여러 사진이 격자로 보이는 갤러리 + 보안 뱃지 1-2개 포함
- **메시지**: "프로젝트별 자동 분류"

### 3. 사진 상세 화면 (증거)
- **촬영 대상**: 스탬프 번인 완료된 사진 + 증거 해시 표시
- **메시지**: "위변조 불가 SHA-256 해시 체인"

### 4. 설정 화면 (커스터마이징)
- **촬영 대상**: 스탬프 설정 (날짜 포맷, 위치, 크기, 투명도)
- **메시지**: "22가지 포맷 · 7곳 위치 · 자유 커스터마이징"

### 5. 보안 모드 (차별점)
- **촬영 대상**: 보안 모드 활성 카메라 + EXIF 차단 표시
- **메시지**: "GPS 완전 차단 · EXIF 스트리핑"

## 촬영 방법 (Android)

### 옵션 A: 실기기 ADB 캡처 (권장)
```bash
# 기기 연결 후 화면 캡처
adb shell screencap -p /sdcard/exacta_01.png
adb pull /sdcard/exacta_01.png ./store_listing/screenshots/

# 또는 한 번에:
adb exec-out screencap -p > store_listing/screenshots/01_camera.png
```

### 옵션 B: 에뮬레이터 (Android Studio)
1. Pixel 6 Pro AVD 생성 (1080×2400)
2. 앱 실행 후 스크린샷 버튼 (카메라 아이콘) 클릭
3. `~/.android/avd/.../screenshots/` 에 저장됨

### 옵션 C: 폰 자체 스크린샷
- 전원 + 볼륨 다운 동시 누르기
- `Pictures/Screenshots/` 폴더에서 PC로 복사
